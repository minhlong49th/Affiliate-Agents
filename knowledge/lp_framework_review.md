# KNOWLEDGE: LP Framework Review (FULL)

## 5. JOEY BABINEAU — PSBCU FRAMEWORK
Full sequence for Review LP: P -> S -> B -> C -> U.
P = Proof (Establish reviewer/brand credibility BEFORE making claims)
S = Story (Third-person framing: "The founder realized...")
B = Benefits (Outcomes only)
C = CTA (Specific action verb)
U = Urgency (Real urgency only)

## 10. HONEST CONS RULE (Review LP)
Joey's model requires 1–2 genuine limitations per review.
Rationale: readers who encounter ONLY positive claims recognize it as marketing. One honest limitation signals credibility.
- Cons must be real limitations, not softened pseudo-cons.

## 11. VERDICT BOX (Review LP)
Joey's signature element. Always appears at the top of Review LP, before the body.
Required elements:
- Star rating (from real source — Trustpilot, Google, Amazon)
- Rating source cited
- "Worth it if:" statement (specific user type)
- "Skip if:" statement (honest exclusion)
- CTA button (linking to affiliate URL)

## AGITATE RULES FOR REVIEW
In three positions: Opener paragraph, Before each benefit section, FAQ questions.
