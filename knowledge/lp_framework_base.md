## 1. LP TYPE → KEYWORD INTENT MAPPING
| Keyword intent | LP type | Traffic source | Expert model |
|---|---|---|---|
| `[brand] coupon / promo code / discount code` | Coupon LP | Google Search branded | Joey |
| `[brand] review / worth it / legit` | Review LP | Google Search branded | Joey |
| `[brand] vs [competitor] / alternative` | Comparison LP | Google Search branded | Both |
| `[problem] solution / best [category]` | Advertorial LP | Facebook / Native / YouTube | Crestani |
| Personalized / quiz / recommendation | Quiz Funnel LP | Facebook / Google broad | Crestani |

**Default rule:** If intent is unclear and traffic source is Google Search branded → use Coupon LP.
If traffic source is Facebook/Native/YouTube → use Advertorial LP.

## 3. WORD COUNT TARGETS
| LP Type | Target range | Notes |
|---|---|---|
| Coupon LP | 300–500 words | Shorter = better. User already knows brand. |
| Review LP | 600–900 words | Long enough to establish credibility, short enough to hold attention |
| Comparison LP | 500–800 words | Table counts toward word count |
| Advertorial LP | 700–1,200 words | 60% must be Problem + Agitate sections |
| Quiz LP (quiz phase) | 150–200 words | Questions only — brevity keeps completion rate high |
| Quiz LP (result page) | 250–350 words | PAS compressed + recommendation + CTA |

## 4. AGITATE BEFORE REVEAL — RULES
Core principle: Write about the user's situation BEFORE introducing the brand or solution.
The brand appears as the answer to a pain that was articulated first.
