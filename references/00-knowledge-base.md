## SECTION 2 — JOEY BABINEAU PSBCU FRAMEWORK (COMPLETE)

### What PSBCU means

**P — Problem:** Acknowledge the pain point the user already has. For coupon searchers: "Stop paying full price." For review searchers: "Not sure which product is right?"

**S — Solution:** Position the LP as the answer. "We have verified coupon codes." "We tested it — read our results."

**B — Benefit:** Quantify the outcome. Always use numbers: "Save 20%", "30-day money back", "Ships in 1–3 days." Avoid vague: "great deal", "good product."

**C — Call to Action:** Tell them exactly what to do. Strong verbs: Reveal, Unlock, Get, See, Read, Compare. Weak verbs to avoid: Click here, Learn more, Visit.

**U — Urgency:** Real urgency only. "Code may expire", "Price changes frequently", "Limited stock." Never fake countdown timers.

### PSBCU distribution in RSA (15 headlines + 4 descriptions)

| Element | Headlines count | Description slots |
|---|---|---|
| P (Problem) | 0–2 (coupon AG) / 2–3 (review AG) | D1 opening sentence |
| S (Solution) | 2–3 | D1 closing |
| B (Benefit) | 3–4 | D2 body |
| C (CTA) | 3–4 | D3 closing + D4 closing |
| U (Urgency) | 2–3 | D4 opening |

### PSBCU for Coupon Intent (AG1) — full example

```
Problem   → (minimal — user already knows they want a coupon)
Solution  → "Verified BuildASoil Promo Code" / "BuildASoil Deals Updated Daily"
Benefit   → "Save Up to 22% Off Today" / "5% to 22% Off All Products"
CTA       → "Reveal Your Discount Code Now" / "Unlock Your BuildASoil Savings"
Urgency   → "Limited Time Coupon Available" / "Code May Expire Anytime"
```

### PSBCU for Product Intent (AG2) — full example


## SECTION 3 — CRESTANI 17-STEP FORMULA (COMPLETE RSA MAPPING)

Crestani's formula is designed for long-form copy (500–1,200 words) but maps to RSA format
when condensed. Use primarily for AG3 (Problem-Aware) and AG4 (Solution-Aware), and for
description enrichment in AG2.

| Step | Full purpose | RSA condensed form | Chars target |
|---|---|---|---|
| 1 Grab Attention | Stop the scroll with hook | H1 PINNED — brand + intent keyword | 25–30 |
| 2 Identify Problem | Name specific pain | H2–H3 — question format | 20–30 |
| 3 Agitate Problem | Amplify before solving | D1 opening — "Most [audience] waste..." | 30–40 |
| 4 Present Solution | Reveal the answer | H4–H5 — "Review Inside" / "Deal Found" | 20–30 |
| 5 Build Credibility | Who tested / discovered | H6–H7 — "X+ Verified Buyers" | 20–28 |
| 6 List Benefits | Outcomes not features | H8–H9 — specific result | 20–30 |
| 7 Social Proof | Numbers, ratings, awards | H10 — "Top Rated" / "X Reviews" | 20–28 |
| 8 Make the Offer | What they get | H11 + D2 — "Up to X% Off" | H: 20–28 |
| 9 Create Urgency | Real reason to act now | H12–H13 — "Limited Time" | 15–25 |
| 10 State Price | Anchor + reveal | H14 — "From $XX" / "Save $XX" | 15–25 |
| 11 Reduce Risk | Guarantee / no-hassle | H15 — "No Signup Needed" | 20–28 |
| 12 Restate Benefits | Top 3 summary | D3 — full sentence | 60–85 |
| 13 Call to Action | Tell them to act | D3 closing — "Click to reveal..." | in D3 |
| 14 Warning/Scarcity | Final push | D4 opening — "Code may expire..." | in D4 |
| 15 P.S. Line | People read H + P.S. first | D4 closing — restate biggest benefit | in D4 |
| 16 Close Loop | Handle objection | D4 — "Worried the code won't work?" | in D4 |
| 17 Final CTA | Repeat action | D4 closing — "Reveal yours now." | in D4 |

### Key Crestani insight on price anchoring (Step 10)
Always anchor high before revealing actual price or savings:
- "Worth $297 — yours for $67" → in affiliate context: "Full price: $89 — with code: $71"
- In headlines: "Save $18 on Every Order" is stronger than "10% Off"
- Numbers create specificity and trust

## SECTION 6 — NEGATIVE KEYWORD STRATEGY PRINCIPLES

### Joey Babineau's core negative principle
"The keyword does half the work — my job is to protect that intent signal by blocking everything
that could dilute it. Every irrelevant click costs money and destroys CVR data."

### Three-tier architecture (Joey standard)
1. **Account-level (Broad):** Universal non-buyers — job seekers, researchers, DIYers
2. **Campaign-level (Phrase):** Category-specific exclusions — competitors, wholesale, B2B
3. **Ad-group-level (Exact):** Cross-contamination prevention between AG1 and AG2

### Crestani's negative approach (broader campaigns)
For Problem-Aware campaigns using Broad Match: add negatives reactively from Search Terms
report after day 3. Do NOT pre-load too many negatives — you'll block discovery traffic.

### Update cadence
- Coupon campaigns (AG1): weekly Search Terms review — high conversion pressure
- Review campaigns (AG2): bi-weekly review
- Problem/Solution campaigns (AG3/AG4): weekly for first month, then bi-weekly

## SECTION 7 — KILL/SCALE DECISION FRAMEWORK

### Joey Babineau's exact thresholds

| Day | Condition | Decision | Action |
|---|---|---|---|
| 1–2 | 0 impressions after 48h | KILL | Bid too low OR LP not indexed |
| 3 | Impressions ≥30, 0 clicks | FIX ONCE | Rewrite H1 — one attempt only |
| 5 | Clicks 0–2 | KILL | Not enough signal |
| 5 | Clicks ≥3, conversions = 0 | WATCH | Keep 3 more days |
| 5 | Clicks ≥3, conversions ≥1 | SCALE | Individual budget $15–20/day |
| 7–8 | WATCH → conversion | CONFIRM SCALE | Increase budget |
| 7–8 | WATCH → still 0 conv, 5–10 clicks | KILL | LP or offer issue |
| Any | Avg CPC > $2.00 | KILL IMMEDIATELY | Too competitive |
| Any | CTR < 0.5% after 100+ impressions | KILL | Ad copy / QS issue |

### Scale mechanics (Joey)
- Tách winner khỏi Shared Budget → set individual budget
- Increase budget: +20% every 48h (not doubling — preserve learning signal)
- Do NOT switch to Smart Bidding until 30–50 conversions in that campaign per month

### Crestani's scale mechanics
- Duplicate winning ad set → new GEO (US → UK → CA → AU)
- Scale horizontal after vertical: same offer → new traffic source
- Budget increase: +20–30% per 48–72h

## SECTION 8 — PLATFORM DIFFERENCES: GOOGLE vs BING

### Why Joey tests on Bing first
- CPC 20–40% lower for same keywords
- Less advertiser competition in affiliate niches
- Older demographic (35+) → higher AOV, more decisive buyers
- Bing coupon intent searcher is more committed than Google equivalent
- Data validation cheaper → migrate winning ad copy to Google with confidence

### Critical Bing-specific setup (must do immediately after campaign creation)
1. **Disable Search Partners** — Campaign Settings → Networks → uncheck Search Partners
   (Search Partners contain click farms and fake traffic sources that destroy CVR data)
2. **Block Audience Network** — Microsoft started injecting search ads into display placements.
   Fix: run Placement Report → filter Audience Network → build exclusion list
3. **Set bids 20–30% lower** than Google equivalents (same keyword, lower CPC)
4. **3-word minimum** — Bing rejects ad assets with fewer than 3 words
5. **No "Click Here"** — Bing explicitly bans non-specific CTAs
6. **No "Official Site"** — Bing bans affiliate use of this claim more strictly than Google

### RSA behavior differences
| Setting | Google | Bing |
|---|---|---|
| Max headlines | 15 | 15 |
| Chars/headline | 30 | 30 |
| Max descriptions | 4 | 4 |
| Chars/description | 90 | 90 |
| Min words | Not specified | 3 words minimum |
| Pinning | Supported | Supported (same syntax) |
| Emoji | Not allowed | Not allowed |
| Exclamation in H | Not allowed | Allowed sparingly |
| "Click Here" | Discouraged | Explicitly banned |
| "Official Site" | Banned for affiliates | Banned for affiliates (stricter enforcement) |
| RSA bulk import | CSV supported | Requires Editor UI for 15 headlines |

## SECTION 9 — AFFILIATE COMPLIANCE RULES (BOTH PLATFORMS)

### What affiliates CAN do in ad copy
- Use brand name in headlines (unless brand's affiliate terms ban PPC)
- Reference discount percentage: "Save 20%", "Up to 22% Off"
- Reference specific products: "BuildASoil Potting Soil 3.0"
- Use coupon-related terms: coupon, promo code, discount code
- Claim "Verified Coupon" or "Working Code" (only if verified)
- Use comparison language: "vs Fox Farm", "Best Price vs Competitors"
- Include guarantee from brand: "Ships with 30-Day Guarantee"

### What affiliates CANNOT do in ad copy
- Claim "Official Site", "Official Store", "Authorized Dealer" (without verification)
- Imply direct brand ownership or manufacturer relationship
- Use fake scarcity (auto-resetting countdown timers)
- Include unsubstantiated claims ("Best product ever", "Guaranteed results")
- Include phone numbers in ad text
- Use exclamation marks in Google headlines
- Use "Click Here" on Bing
- Use emoji
- Reproduce copyrighted taglines verbatim without permission

### FTC disclosure (mandatory on LP — cannot be in ad copy)
Required language near affiliate links: "We earn a commission if you buy through our links."
Must be: visible, near the link, not hidden in footer-only location, readable on all devices.
Note: This cannot go in the ad copy itself — must be on the LP.

## SECTION 10 — QUALITY SCORE OPTIMIZATION PRINCIPLES

Quality Score has 3 components: Expected CTR + Ad Relevance + Landing Page Experience.

### Expected CTR (boost with Joey's approach)
- Pin H1 = exact match keyword → bold when user searches exact query → higher CTR
- Match headline language to search query semantics
- Include benefit in H2 (pinned) → answers "what's in it for me?" immediately

### Ad Relevance (boost with keyword → headline → LP alignment)
- Keyword: "buildasoil coupon code"
- H1: "BuildASoil Coupon Code 2025" ← exact match creates relevance signal
- LP: must contain visible coupon reveal mechanic + product information

### Landing Page Experience (most impactful — often ignored)
- Page must load in < 3 seconds
- Content must match ad promise exactly (no bait-and-switch)
- Must have affiliate disclosure
- Must have privacy policy link
- Must have About/Contact page accessible
- No pop-ups that block content immediately on load
- Mobile-friendly (Google's scoring is mobile-first)

### QS impact on CPC
- QS 10: pays ~50% of competitors' CPC for same position
- QS 7: pays 100% (baseline)
- QS 4: pays ~150% of baseline
- QS 1–3: ads may not show at all

**Actionable:** A well-structured LP with fast load time and matching content can cut CPC by
30–50% compared to a generic bridge page — this directly affects profitability.
